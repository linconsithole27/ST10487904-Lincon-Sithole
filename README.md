Cybersecurity Awareness Chatbot

Description

This project is a console based chatbot built using C sharp in Visual Studio
The chatbot interacts with users and teaches them about basic cybersecurity practices
It simulates a real conversation and gives helpful advice on staying safe online

Purpose

The purpose of this chatbot is to create awareness about cybersecurity in South Africa
Many people are affected by online threats such as phishing scams and weak passwords
This chatbot helps users understand these risks and how to avoid them

Features

Interactive chatbot that responds to user input
Greets the user and asks for their name
Uses the name to personalise the conversation
Provides advice on passwords phishing and safe browsing
Handles invalid or empty input
Displays ASCII art for a better visual experience
Includes a voice greeting using a wav audio file
Simple and structured console user interface

Technologies Used

C sharp programming language
Console application
Visual Studio 2022
GitHub for version control

How to Run

Open the project in Visual Studio
Build the solution to check for errors
Run the application
Listen to the voice greeting
Enter your name when asked
Start asking questions about cybersecurity

Topics Covered

Password safety and strong password creation
Phishing awareness and how to avoid scams
Safe browsing and protecting personal information

Project Structure

Program.cs is used to start the application
Chatbot.cs contains all chatbot logic and methods
Methods are used to organise the code and improve readability
